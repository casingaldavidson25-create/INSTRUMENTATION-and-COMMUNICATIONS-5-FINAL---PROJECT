<h2>📖 PROJECT DESCRIPTION</h2>
<div align="justify">
<b>I.R.O.N. (Intelligent RC Operated Navigator)</b> is a remote controlled car equipped with an <b>RF receiver, transmitter FPV camera, and a Dual Antenna Goggles Monitor</b> for First Person Viewing. An additional <b>Temperature and Humidity sensor</b> is added for the environment monitoring feature to accurately detect the environmental hazards in concern of temperature and humidity in remote areas. The car is controlled using a separate Arduino-based remote controller that uses modulated RF (Radio Frequency) signals typically in the range of <b>2.4Ghz to 2.525Ghz</b> band to transmit and receive data between two Arduinos. The controller uses <b>two (2) joysticks</b>, one for the movement of the car and the other for the movement of the camera for a more accurate environment navigation. An <b>LCD display module</b> is attached in between the joysticks to allow the user to accurately monitor the temperature and humidity status on where the car’s location is.
  
This project is useful for remote inspection, surveillance, and data collection and/or monitoring in environments where direct human presence is difficult and dangerous

The project is divided into <b>two parts</b>. The <b>REMOTE CONTROLLED CAR</b> and the <b>REMOTE CONTROLLER</b> each with different components involved and different codes and configurations.
</div>

<h1 align="center"> <b>REMOTE CONTROLLED CAR</b> </h1>
<h2>🛠️ COMPONENTS USED</h2>

<ul style="display: inline-block; text-align: left;">
  <li>Arduino Nano Microcontroller (1pc.)</li>
  <li>Double BTS7960 43A H-Bridge Motor Driver Module (2pcs.)</li>
  <li>NRF24L01 LNA+PA Transceiver Module (1pc.)</li>
  <li>DHT11 Temperature and Humidity Sensor (1pc.)</li>
  <li>Servo Motor (1pc.)</li>
  <li>1krpm JGA DC Motor Set (4pcs.)</li>
  <li>5.8G 25mW Transmitter FPV AIO Camera (1pc.)</li>
  <li>Rocker Switch (2pcs.)</li>
  <li>18650 Battery (6pcs.)</li>
  <li>7.4V Li-Po Battery (1pc.)</li>
</ul>
